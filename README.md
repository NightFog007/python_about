#### 初始化Flutter项目的admob基本信息
`mkAdmobInfo.py`自动生成admob初始化信息.
* 新建flutter项目后,进入根目录
* 命令行输入`initadmob`,用`cmd+ctrl+t`复制文件到根目录,
* 执行`python3 mkAdmobInfo.py`
